# Senior Frontend Interview 2026 Лучи

Этот репозиторий содержит набор задач для проведения интервью на позицию Senior Frontend Developer в компанию Лучи.

## Стек технологий

- **React 19**
- **TypeScript**
- **Vite**
- **Vitest** (для алгоритмических задач)
- **React Router 7**

## Как запустить

1. Установите зависимости (если еще не установлены):

   ```bash
   npm install
   ```

2. Запустите dev-сервер:

   ```bash
   npm run dev
   ```

3. Откройте в браузере `http://localhost:5173`.
